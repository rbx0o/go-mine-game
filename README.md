# ⛏️ Угольная шахта на Go

Учебный проект, написанный на **Golang**, предназначенный для закрепления концепций второй части полного курса по Go.

Проект представляет собой небольшую игру, в которой игрок управляет предприятием **«Угольная шахта»**, нанимает шахтёров, покупает оборудование и старается пройти игру за минимальное время.

---

## 🎯 Цель проекта

Основная цель проекта — **практическое закрепление материала**, пройденного во **второй части полного курса по Golang**.

В рамках проекта отрабатываются:

- интерфейсы и полиморфизм
- конкурентное программирование (goroutines, channels)
- работа с `context`
- синхронизация данных (`mutex`)
- обработка ошибок
- HTTP-сервер и REST API
- разделение приложения на слои
- базовые принципы проектирования приложений

---

## ⚠️ Дисклеймер

> 🔴 **Важно:**  
> **Этот проект является учебным, а не production-ready решением.**

- Цель проекта — повторение и закрепление материала, а не демонстрация лучших практик

- Основной акцент сделан на понимание механизмов Go, а не на построение идеальной архитектуры

## 🧠 Что делает программа

Игрок управляет предприятием **«Угольная шахта»**.

Предприятие:

- ⛏️ нанимает шахтёров различных классов
- 💰 получает пассивный доход
- 🪨 добывает уголь
- 🛒 покупает оборудование
- 📊 хранит статистику работы
- ⏱️ измеряет время прохождения игры

Цель игры — приобрести всё необходимое оборудование за минимальное время.

---

## 👷 Шахтёры

В игре доступны три типа шахтёров:

### 🪓 Маленький шахтёр

- Стоимость: 5 угля
- Энергия: 30 добыч
- Доход: 1 уголь
- Интервал добычи: 3 секунды

### ⛏️ Нормальный шахтёр

- Стоимость: 50 угля
- Энергия: 45 добыч
- Доход: 3 угля
- Интервал добычи: 2 секунды

### 💪 Сильный шахтёр

- Стоимость: 450 угля
- Энергия: 60 добыч
- Доход: 10 угля
- Интервал добычи: 1 секунда
- С каждой добычей производительность увеличивается

## 🌐 REST API

Взаимодействие с игрой осуществляется через HTTP.

Доступны операции:

### 👷 Шахтёры

- получение информации о классах шахтёров
- найм нового шахтёра
- получение списка активных шахтёров
- фильтрация по типу

### 🏭 Оборудование

- получение списка оборудования
- покупка оборудования
- просмотр приобретённого оборудования

### 📊 Предприятие

- получение статистики
- просмотр текущего баланса
- завершение игры
